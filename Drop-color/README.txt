
Video Color Converter (Windows) — Static Preview Edition
=======================================================

What you get
------------
- No VLC required
- Static image preview generated by ffmpeg (fast, no popup)
- Batch up to 20 files
- Keep original video bitrate if detectable, otherwise 12 Mbps fallback
- Avoids green tint on saturation = 0 (forces yuv420p)
- No per-file popups; single "All done" at the end
- Drag & Drop supported

Folder layout (recommended)
---------------------------
project/
 ├─ video_color_converter_windows.py
 ├─ ffmpeg.exe
 ├─ ffprobe.exe
 ├─ icon.ico            (optional)
 └─ build_windows.bat   (build script)

How to run (from source)
------------------------
1) pip install PyQt5
2) python video_color_converter_windows.py

How to build EXE
----------------
Double-click build_windows.bat
  → dist/Video Color Converter.exe

Notes
-----
- If preview shows "Preview unavailable", check ffmpeg presence.
- Encoding uses original bitrate if ffprobe reports it; else libx264 CRF 23.
- While encoding, UI controls are disabled to prevent thread crashes.
